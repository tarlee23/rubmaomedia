    C++ Code : Stack



    Stack เป็นโปรแกรมเก็บค่า ซึ่งมีการให้ผู้ใช้ระบุค่ามา 5 ค่า และโปรแกรมจะทำการเก็บค่าเพื่อแสดงผลตัวเลขทั้งหมดใน stack ต่อมาจะให้ผู้ใช้ตัดสินใจว่าต้องการนำตัวเลขใน stack ออก 2 ค่าหรือไม่ โดยจะต้องให้คำตอบได้เพียง 2 ค่า คือ yes / no ซึ่งโปรแกรมจะประมวลผล พร้อมทั้งแสดงค่าที่เหลือใน stack และ ตัวเลขที่สูงสุด รวมถึงจำนวนค่าทั้งหมดใน stack
